# vue_cli_webpack生成的项目，只含了router模块

## 运行说明

``` bash
# 安装依赖
npm install
# 启动本地热更新服务localhost:8080
npm run dev
# 生产环境构建(含压缩)
npm run build
# 构建报告
npm run build --report
```
